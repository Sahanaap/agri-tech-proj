# Agri Tech Project

## About The Project
Agri Tech Project is a web-based application designed to provide agricultural-related solutions and information through a simple and user-friendly interface. The project uses frontend technologies along with a Node.js backend server.

## Features
- Agriculture information interface
- Interactive web design
- Fast and responsive UI
- Backend server using Node.js
- Client-server communication

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

 ## Project Structure
 agri-tech-proj/
    index.html # Main webpage
    style.css # Styling file
    script.js # Frontend logic
    server.js # Backend server
    package.json # Project dependencies
    package-lock.json # Dependency lock file

###  Prerequisites
Make sure you have installed:
  Node.js
   npm

### Installation
Clone the repository 
   git clone https://github.com/Sahanaap/agri-tech-proj.git
   
Navigate into project folder
    cd agri-tech-proj

Install dependencies
    npm install


### Run The Project
Start the server: node server.js
    Open your browser and go to:
          http://localhost:3000
